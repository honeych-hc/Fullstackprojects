Stone Paper Scissors Game
A simple implementation of the classic game using HTML, CSS, and JavaScript
Game Description
A fun and interactive game of Stone Paper Scissors, where you can play against the computer.

Features
- User-friendly interface
- Computer-generated random choices
- Win/loss tracking
- Simple and intuitive design

Code Structure
- index.html: Game structure and layout
- style.css: Game styling and design
- app.js: Game logic and functionality

How to Play
1. Open index.html in a web browser
2. Click on Stone, Paper, or Scissors to make your choice
3. The computer will generate a random choice
4. The result will be displayed on the screen

JavaScript Logic
- getComputerChoice(): Generates a random choice for the computer
- playGame(): Compares user and computer choices, determines winner
- updateScore(): Updates the score display
